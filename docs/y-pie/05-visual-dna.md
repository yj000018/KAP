# Visual DNA and Personal Knowledge Genome

## Definition

Visual DNA is the evolving personal model of recurring visual preferences, motifs, structures, emotional signatures and creative tendencies inferred from the user's visual corpus and validated choices.

It is not a single similarity score. It is a versioned, explainable, multi-dimensional profile.

## Relationship to the Personal Knowledge Genome

The Personal Knowledge Genome is a KAP-level cross-modal model spanning visual, textual, behavioral, spatial, temporal and relational knowledge. Visual DNA is its visual facet.

```text
Personal Knowledge Genome
├── Visual DNA
├── Creative DNA
├── Cognitive patterns
├── Memory patterns
├── Relationship patterns
└── Behavioral patterns
```

Y-PIE computes visual traits and evidence. KAP integrates them into the broader personal model.

## Visual DNA dimensions

- palette preferences;
- luminance and contrast tendencies;
- compositional structures;
- preferred geometries;
- abstraction level;
- figurative versus symbolic balance;
- recurring objects and environments;
- archetypes and symbols;
- emotional spectrum;
- atmosphere and materiality;
- stylistic families;
- project affinities;
- curation behavior;
- rejection behavior;
- evolution over time.

## Personal similarity

A future `Yannick-likeness` score may estimate alignment with the current Visual DNA. It must always expose contributing traits and uncertainty. It must not be interpreted as artistic quality.

## Evolution model

Visual DNA is computed by period and globally. It must reveal:
- stable traits;
- emerging traits;
- declining traits;
- creative ruptures;
- cyclic returns;
- project-specific sublanguages;
- unexplored but adjacent territory.

## Learning signals

Strong explicit signals:
- manual ratings;
- project assignment;
- keep/delete decisions;
- selected hero image;
- collection inclusion;
- explicit critique.

Weak implicit signals:
- favorites;
- editing effort;
- sharing;
- revisits;
- repeated generation;
- time spent comparing.

Implicit signals must never override explicit corrections.

## Outputs

- Visual DNA profile;
- creative-period map;
- recurring archetype report;
- palette and geometry atlas;
- personal similarity explanation;
- emerging-language alerts;
- underexplored creative directions;
- bridge concepts between projects.

## Guardrails

Visual DNA is descriptive, not deterministic. It must support creative divergence rather than push all new work toward past preferences.
