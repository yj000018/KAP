# Y-PIE Architecture Decision Records

Use ADRs for durable choices affecting ownership, contracts, storage, models, privacy, deployment or irreversible migration.

## Format

```markdown
# ADR-NNN — Title

- Status: proposed | accepted | superseded | rejected
- Date: YYYY-MM-DD
- Owners:

## Context
## Decision
## Alternatives considered
## Consequences
## Migration and rollback
## Security and privacy impact
## References
```
