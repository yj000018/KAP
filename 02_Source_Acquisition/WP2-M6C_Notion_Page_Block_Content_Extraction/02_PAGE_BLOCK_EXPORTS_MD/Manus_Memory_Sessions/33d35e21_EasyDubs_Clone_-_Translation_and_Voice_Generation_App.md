Failed / suboptimal:

- Initial mock translation service showing placeholders
- Speech recognition returning same phrase repeatedly

Discoveries:

- Need for comprehensive testing of mock services
- Importance of varied mock data for realistic testing
- Value of permanent deployment for consistent access
