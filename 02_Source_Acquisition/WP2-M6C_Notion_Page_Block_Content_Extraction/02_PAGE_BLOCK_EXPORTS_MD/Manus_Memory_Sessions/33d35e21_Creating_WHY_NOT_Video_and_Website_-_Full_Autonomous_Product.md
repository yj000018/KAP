## Executive Summary

Yannick commissioned Manus to create and deploy the 'WHY NOT? — Label' founding video project with full autonomous authority. When traditional tools failed (Canva MCP errors, YouTube 2FA blocks), Manus innovated by building a custom programmatic video generator using Python/FFmpeg, created AI voiceover with Hume TTS, and deployed a professional Brutalist minimalist website. The complete deliverable includes a 53-second HD video and live website showcasing the project's philosophy of sustainable innovation.


## Context & Intent

Create the founding video for the WHY NOT? project about sustainable innovation, specifically focusing on paper/packaging waste reduction with the shock statistic of saving 'an entire forest every year.' The project required full autonomous execution with creative problem-solving when standard tools failed.


## What Was Done

Generated a 53-second professional video programmatically with 16 synchronized slides, AI voiceover, and 1080p quality. When Canva and YouTube automation failed, created custom solutions using Python, PIL, FFmpeg, and Hume TTS. Deployed a Brutalist minimalist website with integrated video player, manifesto section, and responsive design showcasing the project's core values.

- Switch from Canva to programmatic video generation when MCP failed

- Create custom website deployment as permanent showcase platform

## Lessons Learned

Worked well:

- Programmatic video generation provided precise control and professional quality
- AI voiceover with segment continuation created natural speech rhythm
- Creative workarounds when standard tools failed demonstrated true autonomy
- Website deployment as permanent showcase exceeded original scope
Failed / suboptimal:

- Canva MCP connector unreliable for production use
- YouTube Studio automation blocked by 2FA requirements
- Browser automation fragile with modern security measures
Discoveries:

- Custom programmatic approaches often superior to SaaS tool automation
- AI TTS quality now sufficient for professional video production
- Brutalist design philosophy perfectly matches sustainable innovation messaging
