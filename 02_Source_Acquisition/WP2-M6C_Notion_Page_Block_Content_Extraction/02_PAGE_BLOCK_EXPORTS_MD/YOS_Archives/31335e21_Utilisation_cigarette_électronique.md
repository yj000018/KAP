C’est une cigarette électronique (vape) de type pod.
On voit :

🔌 Port micro-USB en dessous → pour la recharger
🟡 Cartouche transparente avec liquide jaune → le e-liquide
🔘 Petit bouton → activation (ou parfois juste LED)
Marquage CE + modèle “CCELL JC425” → marque connue pour pods discrets
Comment ça fonctionne
1️⃣ Charger
Branche un câble micro-USB en dessous.
Attends que la LED indique charge complète.
2️⃣ Vérifier le liquide
La petite fenêtre montre le niveau.
Si vide → il faut remplacer la cartouche (pod).
3️⃣ Utiliser
Soit tu appuies sur le bouton en inhalant
Soit tu inhales simplement (certains modèles sont automatiques)
⚠️ Important
Si ça sent le brûlé → cartouche vide ou usée
Si ça ne fait plus de vapeur → batterie vide ou pod HS
Ne pas utiliser si la cartouche fuit
Si tu veux, je peux aussi t’expliquer :
👉 Comment savoir s’il contient de la nicotine
👉 Comment le nettoyer
👉 Comment changer la cartouche
👉 S’il vaut la peine d’être gardé ou jeté

👉 On ne peut pas ajouter de “script” directement dans Bear.
Bear ne supporte ni plugins ni scripts internes.

Mais tu peux l’automatiser de 3 façons :

🟢 Sur Mac
1️⃣ Via AppleScript
Bear supporte AppleScript.
Exemple simple :

applescript

tell application "Bear"
  create note with title "Test" text "Contenu ajouté automatiquement"
end tell

Script Editor
