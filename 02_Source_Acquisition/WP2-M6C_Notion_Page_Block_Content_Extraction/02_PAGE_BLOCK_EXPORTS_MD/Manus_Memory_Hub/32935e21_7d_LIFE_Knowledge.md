### Working title

7d LIFE — Flower Coaching App


### Core concept

The app models life through 7 chakra petals, each representing a life domain.


### Fixed 7-petal model

1. Root — red — body, safety, money, home, routines
1. Sacral — orange — emotions, pleasure, creativity, sensuality, play
1. Solar Plexus — yellow — confidence, willpower, work, boundaries, power
1. Heart — green — love, relationships, compassion, self-love
1. Throat — blue — communication, truth, self-expression, honesty
1. Third Eye — indigo — clarity, intuition, mindset, vision, focus
1. Crown — violet — meaning, spirituality, purpose, alignment

### Scoring logic

Per-petal score: 0-10

State labels:

- 0-3: Crisis / Blocked
- 4-5: Unstable / Wobbly
- 6-7: Stable / Functional
- 8-10: Thriving / Radiant

### Derived metrics

- 7D Life Score = average of last known scores
- Balance Score = highest minus lowest score

### Main flows

Onboarding, Baseline Assessment, 90-Day Goal Setup, Dashboard, Petal Detail, Weekly Tracker, 90-Day Review


### Technical direction

React + TypeScript, Tailwind, API backend, SQL persistence, modular architecture
