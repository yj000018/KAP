**OAuth scope:** `create_task` or `manage_all_tasks` — see the [OAuth2](https://open.manus.ai/docs/v2/oauth) guide. With `create_task` scope, can only access tasks created by this OAuth app.
</Tip>

<Tip>

</Tip>
