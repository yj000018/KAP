<Tip>
**OAuth scope:** `create_task` or `manage_all_tasks` — see the [OAuth2](https://open.manus.ai/docs/v2/oauth) guide. With `create_task` scope, can only access tasks created by this OAuth app.
</Tip>

<Warning>
This action is permanent. If the task is still running, use [task.stop](https://open.manus.ai/docs/v2/task.stop) to stop it first. Agent-related tasks (e.g. agent main tasks and subtasks) cannot be deleted.
</Warning>
