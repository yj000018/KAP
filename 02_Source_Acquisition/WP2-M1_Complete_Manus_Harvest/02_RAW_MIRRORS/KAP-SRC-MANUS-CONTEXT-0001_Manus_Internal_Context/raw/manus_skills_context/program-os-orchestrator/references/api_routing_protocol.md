# API Routing Protocol

This protocol defines how to route tasks to specialized LLMs (like Claude) for review or specialized generation within a Program OS workflow.

## When to Use Claude

- **Review and Critique**: Use Claude to review drafts generated by ChatGPT or Manus.
- **Architectural Audits**: Use Claude for deep, structural analysis of complex ontologies or frameworks.
- **Tone and Style**: Use Claude when specific philosophical, academic, or humanistic tones are required.

## Model Selection

- **Standard Review**: `claude-sonnet-4-6` (Fast, capable, good for standard document review).
- **Heavy Architectural Review**: `claude-opus-4-8` (Slower, but superior for complex reasoning and deep structural analysis).

## Implementation Pattern

Always use the official `anthropic` Python SDK. Ensure the `ANTHROPIC_API_KEY` is set in the environment.

```python
import os
from anthropic import Anthropic

def run_claude_review(draft_content, review_prompt, model="claude-sonnet-4-6"):
    api_key = os.environ.get("ANTHROPIC_API_KEY")
    if not api_key:
        raise ValueError("ANTHROPIC_API_KEY environment variable not set.")
        
    client = Anthropic(api_key=api_key)
    
    try:
        response = client.messages.create(
            model=model,
            max_tokens=4096,
            system="You are a Chief Architect reviewing a document for structural integrity, tone, and alignment with core principles.",
            messages=[
                {"role": "user", "content": f"{review_prompt}\n\nDocument Draft:\n{draft_content}"}
            ]
        )
        return response.content[0].text
    except Exception as e:
        print(f"Claude API Error: {e}")
        # Log error and implement fallback or fail gracefully
        return None
```
