# Checklist de Especificidade para Prompts

Usar esta checklist para avaliar e completar prompts antes de executar no Agent Mode.
Cada item marcado como faltante deve ser coletado do usuário via Chat Mode (gratuito).

## Checklist Universal (Todas as Tarefas)

- [ ] **Objetivo claro**: O que exatamente deve ser entregue?
- [ ] **Formato de saída**: Arquivo, site, documento, código, etc.?
- [ ] **Idioma**: Em que idioma deve ser o resultado?
- [ ] **Público-alvo**: Para quem é destinado?
- [ ] **Referências**: Há exemplos ou modelos a seguir?
- [ ] **Restrições**: Há limitações de tempo, tamanho, tecnologia?

## Checklist para Desenvolvimento Web

- [ ] **Textos exatos**: Todo texto visível está definido entre aspas?
- [ ] **Cores**: Paleta definida com códigos hex (#RRGGBB)?
- [ ] **Fontes**: Família tipográfica especificada?
- [ ] **Layout**: Estrutura de seções descrita ou referência visual?
- [ ] **Responsividade**: Comportamento mobile/tablet/desktop definido?
- [ ] **Estados de erro**: Mensagens de erro para formulários/ações?
- [ ] **Estados de sucesso**: Feedback visual para ações bem-sucedidas?
- [ ] **Navegação**: Estrutura de páginas e links definida?
- [ ] **Interações**: Comportamento de hover, click, scroll?
- [ ] **Imagens/Ícones**: Fontes de assets definidas?

## Checklist para Pesquisa/Relatório

- [ ] **Escopo**: Período temporal, região geográfica, setor?
- [ ] **Profundidade**: Visão geral vs. análise detalhada?
- [ ] **Fontes**: Tipos de fontes preferidas (acadêmicas, jornalísticas, etc.)?
- [ ] **Estrutura**: Seções ou tópicos obrigatórios?
- [ ] **Dados**: Tabelas, gráficos ou estatísticas necessárias?
- [ ] **Comparações**: Empresas, produtos ou conceitos a comparar?

## Checklist para Análise de Dados

- [ ] **Dados de entrada**: Formato e localização dos dados?
- [ ] **Métricas**: KPIs ou métricas específicas a calcular?
- [ ] **Visualizações**: Tipos de gráficos desejados?
- [ ] **Período**: Janela temporal da análise?
- [ ] **Agrupamentos**: Como segmentar os dados?
- [ ] **Formato de saída**: Dashboard, relatório, planilha?

## Checklist para Conteúdo/Texto

- [ ] **Tom de voz**: Formal, casual, técnico, persuasivo?
- [ ] **Extensão**: Número de palavras/páginas?
- [ ] **Estrutura**: Tópicos ou outline definido?
- [ ] **SEO**: Palavras-chave a incluir?
- [ ] **Call-to-action**: Ação desejada do leitor?
- [ ] **Referências**: Fontes a citar ou evitar?

## Checklist para Automação

- [ ] **Trigger**: O que dispara a automação?
- [ ] **Frequência**: Com que frequência deve executar?
- [ ] **Inputs**: Quais dados de entrada?
- [ ] **Outputs**: Qual o resultado esperado?
- [ ] **Erros**: Como lidar com falhas?
- [ ] **Dependências**: Serviços ou APIs necessários?
