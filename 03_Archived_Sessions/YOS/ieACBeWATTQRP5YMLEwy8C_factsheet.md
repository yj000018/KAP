### 💡 Key Insights
- **Ghost** retenu comme MVP stack (site + newsletter, 0% frais, ownership total) si Yannick veut éviter la complexité de gérer deux outils séparés
- **Medium** = présence/front-end texte secondaire, pas de publication photo — rôle similaire à Threads pour le texte
- **Architecture 2 pôles** confirmée : CIVILISATIONS (IA, futur, société, matières/pierres) + CONSCIENCE/MAGNÉTI (intériorité, spiritualité, présence)
- **Fédération sous "Y"** comme marque personnelle ultime envisagée — les deux sites seraient Humanity 2.0 et Présence/Conscience
- **"The Witness"** rejeté car trop passif — direction préférée : transformation active, éveil, révolution, singularité intérieure/extérieure
- **Framer** jugé trop technique par Yannick ; **Lovable** préféré pour la partie constructeur de site
- Le lien entre les deux pôles thématiques est volontairement explicite dans l'architecture (montrer la connexion conscience ↔ civilisation)
