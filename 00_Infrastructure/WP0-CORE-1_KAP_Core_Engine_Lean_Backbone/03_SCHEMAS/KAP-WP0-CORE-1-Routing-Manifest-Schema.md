# Routing Manifest Schema

**Sprint:** WP0-CORE-1

Defines where an acquired item should be sent within the yOS ecosystem.

Allowed `scope` values:
- `YOS_SELF_KNOWLEDGE`
- `PROJECT_KNOWLEDGE`
- `YOUNIVERSE`
- `TELEMETRY`
- `ARCHIVE_ONLY`
- `MEM0_CANDIDATE_LATER`
- `OPERATIONAL_NOISE`
- `RESTRICTED_SECRET_SURFACE`
