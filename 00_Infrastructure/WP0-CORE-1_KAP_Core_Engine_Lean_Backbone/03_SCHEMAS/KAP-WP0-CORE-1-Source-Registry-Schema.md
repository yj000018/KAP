# Source Registry Schema

**Sprint:** WP0-CORE-1

Defines the structure for tracking any knowledge source in KAP.

Allowed `source_status` values:
- `DISCOVERED`
- `INVENTORIED`
- `BASELINE_IN_PROGRESS`
- `BASELINE_CAPTURED`
- `BASELINE_VERIFIED`
- `DELTA_READY`
- `FROZEN`
- `DEFERRED`
- `BLOCKED`
- `EXCLUDED`
- `NON_CORPUS`
- `REOPEN_REQUIRED`
