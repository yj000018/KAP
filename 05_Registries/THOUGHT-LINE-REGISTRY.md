## Maturity Levels

```text
seed → exploratory → emerging_pattern → candidate_architecture → validated_architecture → canonical → deprecated
```

## Rules

1. A Thought Line is never deleted — only deprecated
2. Maturity upgrades require gate evidence
3. Current Best Synthesis is generated only at `candidate_architecture` or above
4. Review triggers: new contradicting source, 30+ days without update, explicit architect request
5. Source corpus Thought Lines track acquisition state, not just intellectual evolution
